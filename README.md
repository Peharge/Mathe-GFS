## Installation

### Instalation:

### 1. Klone das Repository:
```bash
git https://github.com/Peharge/Woyzy
   ```
### 2. Navigiere in das Projektverzeichnis:
```bash
cd Woyzy
```
### 3. Installation von Manim :
```bash
pip install manim
```
### 4. Test:
```bash
manim -p -r 3840,2160 main1.py ContinuousMotion
```
